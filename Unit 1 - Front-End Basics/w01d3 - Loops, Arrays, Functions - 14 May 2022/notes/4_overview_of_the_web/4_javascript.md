## What is Javascript?
Javascript provide us the ability to have interactions on our website. Almost 99.99% of websites now have some sort of interactivity. Some things we can do with Javascript:

1. When a user click a button, pop up a subscribe form
2. Modify HTML elements when user click a button
3. Interact with backend services without reloading the page
and many more...
